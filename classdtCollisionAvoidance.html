Detailed Description

Defines a behavior for collision avoidance.

The agents use this behavior in order to change their velocity to avoid obstacles.

float dtCollisionAvoidance::sampleOriginScale

Samples origin scale.

The center of the first-level pattern is chosen to be the desired velocity scaled by this value. The larger it is, the more the agent will accelerate to avoid collisions.

Remarks
Default value is 0.4.
unsigned char dtCollisionAvoidance::sampleRingsCount

The number of samples rings per level.

Remarks

Remarks

float dtCollisionAvoidance::weightCurrentAvoidanceSide

The weight of the current avoidance side.

Increase to prefer candidates that will keep the currently chosen avoidance side. In practice, if the agent is already passing to the left of its neighbors this weight will favor candidates that keeps this side of avoidance.

Remarks
Default value is 0.75.
float dtCollisionAvoidance::weightCurrentVelocity

The weight of the current velocity.

Increase to prefer candidates close to the current velocity.

Remarks
Default value is 0.75.
float dtCollisionAvoidance::weightDesiredVelocity

The weight of the desired velocity. 

- the distance to the desired velocity;
- the distance to the current velocity;
