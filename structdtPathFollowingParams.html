Detailed Description

Parameters for the path following behavior.

Warning
The dtPathFollowingParams::init() must be called before the parameter is used.

Initializes the corridor.

This method MUST be called before using the path following behavior with this parameter.

Parameters
[in]maxPathResultThe maximum number of polygons that can be stored in a corridor.
[in]positionThe position of the agent. Used to determine which polygon it is on.
[in]queryUsed to access the navigation mesh query in order to get the polygon the agent is on.
